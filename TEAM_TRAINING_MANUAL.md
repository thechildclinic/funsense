## 🚀 **GETTING STARTED**

### **System Requirements**
- **Device**: Tablet, laptop, or smartphone
- **Browser**: Chrome, Firefox, Safari, Edge (latest versions)
- **Internet**: Stable connection required
- **Camera**: Built-in or external USB camera
- **Permissions**: Camera and microphone access

### **First Time Setup**
1. **Open the application** in your browser
2. **Allow camera permissions** when prompted
3. **Test camera functionality** before first screening
4. **Bookmark the site** for easy access
5. **Install as PWA** (optional) for offline capability

## 📋 **SCREENING WORKFLOW**

### **Step 1: Student Identification**
```
1. Enter student ID manually
2. Fill in basic information:
   - Name
   - Age
   - Gender
   - Pre-existing conditions
3. Click "Start Screening"
```

### **Step 2: Anthropometry**
```
Height Measurement:
1. Position student against measuring station
2. Capture image with ruler visible
3. Tap top and bottom of student's head
4. Tap ruler reference points
5. System calculates height

Weight Measurement:
1. Have student step on scale
2. Capture clear image of display
3. AI reads the weight value
4. Verify and correct if needed

BMI Calculation:
- System automatically calculates
- AI provides interpretation
- Review results before proceeding
```

### **Step 3: Specialized Imaging**
```
ENT Examination:
1. Select examination type (ear/nose/throat)
2. Position camera appropriately
3. Capture clear images
4. AI analyzes for abnormalities
5. Add nurse notes

Dental Examination:
1. Position for oral cavity view
2. Capture images or video
3. AI analyzes dental health
4. Document findings
```

### **Step 4: Vital Signs**
```
Device Vitals:
1. Take measurements with medical devices
2. Capture clear images of displays
3. AI reads values via OCR
4. Verify accuracy

Face Wellness (Simulated):
1. Capture frontal face image
2. AI provides general observations
3. For educational purposes only

Stethoscope (Simulated):
1. Position stethoscope correctly
2. Capture placement image
3. Educational simulation only
```

### **Step 5: Review & Export**
```
1. Review all collected data
2. Add final nurse observations
3. Generate AI summary
4. Export report (JSON/Print)
5. Mark screening complete
```

## 🛠️ **TROUBLESHOOTING QUICK REFERENCE**

### **Camera Issues**
```
Problem: Camera not working
Solutions:
1. Check browser permissions
2. Refresh the page
3. Try different browser
4. Check camera hardware
5. Restart device
```

### **AI Analysis Errors**
```
Problem: AI not responding
Solutions:
1. Check internet connection
2. Retry the analysis
3. Verify image quality
4. Contact IT support
```

### **Data Not Saving**
```
Problem: Progress lost
Solutions:
1. Check browser storage settings
2. Don't use incognito mode
3. Clear browser cache
4. Try different browser
```

### **Performance Issues**
```
Problem: App running slowly
Solutions:
1. Close other browser tabs
2. Restart browser
3. Check internet speed
4. Clear browser cache
5. Use recommended browsers
```
